# Interview Prep App

Application for **personalized interview preparation** - it helps you practice conceptual, behavioral, and coding questions tailored to your **resume** and a **job description**, tracks your **strengths, weaknesses, and improvement areas**, and provides AI-generated feedback on your answers.

--------------------------------------------------

## Features

### Prep Hub
- Mode-wise question practice: **Conceptual, Behavioral, Coding**
- Questions generated based on **resume + job description**
- AI-generated **explanations and hints**
- Tracks top **strengths, weaknesses, and areas for improvement** (last 5 sessions)

### Mock Interview
- Practice interview-style questions in a **continuous flow**
- Submit answers and receive **AI-generated feedback**
- Configurable **difficulty** and **interview mode**
- **Next Question** button to move through multiple questions

### Progress Dashboard
- Visual summary of your **Prep Hub performance**
- Highlights **Top 3 Strengths, Top 3 Weaknesses, Top 3 Areas for Improvement**
- Color-coded feedback for intuitive understanding

---------------------------------------------------
